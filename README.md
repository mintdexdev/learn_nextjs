# Learn Nextjs 
Learnt about nextjs using this repo


To get you the local copy of the repo:
```
git clone https://github.com/mintdexdev/learn_nextjs.git
cd learn_nextjs
```

To run the develpment server:
```
cd project_name
&&
npm i
&&
npm run dev
```

To build and preview:

```
npm run build 
&& 
npm run start
```
